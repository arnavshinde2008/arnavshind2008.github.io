# 🐍 Snake Game

A classic Snake Game built using **HTML**, **CSS**, and **JavaScript**. Control the snake, eat food to grow longer, and avoid hitting the walls or yourself!

---



Example:

![Snake Game Screenshot](screenshot.png)

---

## 🚀 Features

- Smooth snake movement
- Keyboard controls (Arrow Keys)
- Random food generation
- Score counter
- Snake grows after eating food
- Wall collision detection
- Self collision detection
- Restart button
- Simple and responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/arnavshinde2008/Snake-Game.git
```

2. Open the project folder.

3. Double-click `index.html` or open it with your preferred web browser.

No installation or additional software is required.
